#WiFiMSXSM

WiFi Library for MSX-SM
Copyright (c) 2019 Oduvaldo Pavan Junior ( ducasp@ gmail.com )
All rights reserved.

Functions that allow your program to access the WiFi module of your MSX-SM.

The header file is a small documentation of the functions, look at SNTP or
TELNET clients source code for pratical examples of usage.

Requires SDCC and Fusion-C library to compile
