ディスクイメージ関連ツールを1チップMSXでご使用になる方への補足事項

■ ディスクイメージ関連ツール
・EP      .COM : FD emulate table setter for MEGA-SCSI
・ESET    .COM : MEGA-SCSI setup utility
・FDCOPY  .COM : Sector to Sector copy tool
・FDLOAD  .COM : File to Sector convert tool
・FDSAVE  .COM : Sector to File convert tool
・MGLOAD  .COM : MegaROM loader for MEGA-SCSI
・MGSAVE  .COM : MegaROM to File convert tool

これらのツールの基本的な使用方法に関しては、1チップMSX取扱説明書をご参照下さい。

より高度な使用方法に関しては、1チップMSX付属CD-ROMの\Tools\DiskImage\reference
の各アーカイブに同梱されているドキュメントをご参照下さい。
